## ✨ What Makes This Portfolio Extraordinary

This isn't a template. Every pixel was deliberately crafted from scratch.

| Feature | Description |
|---|---|
| 🎬 **Cinematic Loader** | Custom animated loader that orchestrates the entire page entrance sequence |
| 🪄 **Staggered Hero Reveal** | 8 independent elements fade in with carefully timed delays after loader completes |
| 🌌 **Particle Network Canvas** | 90-particle WebGL-like canvas with dynamic connection lines — fully custom, zero library |
| ⌨️ **Typewriter Engine** | Cycles through 8 role titles with natural typing/deleting rhythm |
| 🎯 **Custom Cursor System** | Dual-layer magnetic cursor (dot + ring) with smooth lag interpolation via `requestAnimationFrame` |
| 🔮 **Physics Skills Arena** | Interactive ball-physics simulation — drag, throw, and collide your tech skills |
| 📜 **Scroll-Driven Reveals** | `IntersectionObserver` triggers elegant slide-up animations as you scroll |
| 🧲 **Glassmorphism Cards** | Backdrop-filter blur cards with radial-gradient hover glow |
| 👩‍💻 **CSS Girl-Coder Avatar** | 100% CSS-drawn animated character — no SVGs, no images, just pure CSS divs |
| 📱 **Mobile-First Responsive** | Full hamburger nav, fluid layouts, touch-optimized at every breakpoint |

## 🛠️ Tech Stack

### Frontend Core

```
React 19        →  Component architecture, hooks (useState, useEffect, useRef)
Vite 8          →  Lightning-fast HMR dev server, optimized production builds
Vanilla CSS     →  Custom design system, zero framework overhead
```

### Design System

```
Bebas Neue      →  Display headings (section titles, hero name)
JetBrains Mono  →  Terminal / code aesthetic text
Satoshi         →  Body copy and descriptions
```

### CSS Custom Properties (Token System)

```css
--bg: #020409          /* Deep space black     */
--green: #00ff9d       /* Signature neon green */
--cyan: #00d4ff        /* Accent cyan          */
--purple: #a855f7      /* AI/ML accent         */
--orange: #ff6b35      /* Highlights           */

--head: 'Bebas Neue'
```

### Browser APIs Used (Zero Dependencies)

```
Canvas API              →  Particle network, skills physics simulation
requestAnimationFrame   →  Cursor interpolation, particle animation loop
IntersectionObserver    →  Scroll-triggered reveal animations
Web Animations API      →  CSS transitions + transform-based effects
```

## ⚡ Getting Started

### Prerequisites

```bash
node  >= 18.0.0
npm   >= 9.0.0
```

### Installation & Run

```bash
# Clone the repository
git clone https://github.com/Vidhya-Majee/portfolio.git

# Navigate into the project
cd portfolio-react

# Install dependencies
npm install

# Start development server
npm run dev
```

Open **[http://localhost:5173](http://localhost:5173)** in your browser.

### Build for Production

```bash
# Create an optimized production build
npm run build

# Preview the production build locally
npm run preview
```

### Lint

```bash
# Run Oxlint (ultra-fast Rust-based linter, 50-100x faster than ESLint)
npm run lint
```

## 📁 Component Architecture

```
App (State: started, navShow)
 │
 ├── Cursor          ← Global: runs independently, intercepts all interactions
 ├── Loader          ── fires onDone() → sets started=true, navShow=true
 │
 ├── Navbar          ← show prop animates in after loader
 ├── Hero            ← started prop controls staggered entrance (8 elements)
 │
 ├── Expertise       ┐
 ├── Skills          │  All use IntersectionObserver
 ├── Projects        │  for scroll-triggered reveals
 ├── Experience      │
 ├── Certifications  │
```
